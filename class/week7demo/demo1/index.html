1. Resize the browser window. A container with "flex-wrap: nowrap;" will never wrap its items.

2. Try changing your CSS from "flex-wrap: nowrap;" to "flex-wrap: wrap;". What happens?

3. Play around. Change how many boxes appear. Add content within them. Add text to the page, change the CSS.

4. Is there a way for you to modify this code to be useful to your interview project?

5. Look at the CSS closely. Try commenting out the CSS that's currently active, and activating the CSS that's dormant. How does that change the way the content displays?
